# Aglais Extended Quantum VM

Aglais XQVM is a unified virtual machine for quantum computing. It provides a single bytecode target and toolchain for expressing quantum problem specifications, which can then be compiled and executed across heterogeneous quantum hardware backends — including quantum annealers, gate-based chips, and other quantum computing architectures.

## Goals

- **Hardware-agnostic**: write a problem once, run it on any supported quantum backend
- **Unified bytecode**: a common intermediate representation decoupled from any specific hardware model
